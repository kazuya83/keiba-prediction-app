# フェーズ09 レースデータAPI実装

## 目的
フロントエンドがレース情報を取得できるよう REST API を整備し、データ提供の基盤を固める。

## 主なタスク
- `GET /api/races` の実装（フィルタリング、ページング、並び替え）
- `GET /api/races/{race_id}` の実装（詳細情報、関連データの取得）
- キャッシュ・クエリ最適化によるレスポンス改善
- 認証・権限チェックの組み込み
- API ドキュメントとサンプルレスポンス整備

## 成果物
- レース関連 API コード
- OpenAPI 仕様更新
- API テストケース（単体／統合）
