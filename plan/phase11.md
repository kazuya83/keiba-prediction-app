# フェーズ11 予測APIと履歴管理実装

## 目的
ユーザーが予測を実行し履歴を確認できるバックエンド機能を整備し、精度計測の基盤を作る。

## 主なタスク
- `POST /api/predictions` の実装（非同期実行・キュー処理検討を含む）
- 予測結果保存・履歴取得 API の実装（一覧・詳細・比較）
- 的中率、回収率、平均確率などの指標算出ロジック実装
- モデル再学習トリガーとの連携検証
- 監査ログ・エラーハンドリング実装

## 成果物
- 予測関連 API コード
- 指標算出ユーティリティ
- API テストと精度検証レポート

- `requirements.md` 2.3 予測機能、2.5 予測履歴管理、7.3 予測API
- 予測完了時間 1 分以内という制約を満たすこと
