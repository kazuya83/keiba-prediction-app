# Frontend

React + TypeScript によるフロントエンドアプリケーションです。pnpm を利用して依存関係を管理します。

## セットアップ

```bash
pnpm install
pnpm dev
```

## 推奨ディレクトリ構成

```
src/
  components/
    common/
    domain/
  features/
  pages/
  hooks/
  lib/
  stores/
  styles/
```

## 使用予定ツール

- React 18
- Vite
- TypeScript (strict)
- ESLint + Prettier
- Vitest + React Testing Library
